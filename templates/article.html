Controlling 28byj-48 Stepper Motor With an ESP32

INTRODUCTION

Controlling a 5V stepper motor with an ESP32 is a simple project that’s great for learning about electronics and robotics. Stepper motors are useful for tasks that need precise movements, like rotating a dial or moving a platform. The ESP32 is a powerful microcontroller with built-in Wi-Fi and Bluetooth, making it a great choice for modern projects. In this tutorial, we’ll cover how to connect a 5V stepper motor to an ESP32, wire the circuit, and write some basic code to control the motor. By the end, you’ll be ready to use this setup in your own projects.

Materials used in this project:

28BYJ-48 Stepper Motor-5V

ULN2003

ESP32 Dev Module

5-12v Power Source

Breadboard

Jumper Wires

const int steps_per_rev = 2048;
    
Stepper Smol_Stepper(steps_per_rev, 19, 5, 18, 17);
    
void setup() {
    Smol_Stepper.setSpeed(10);
    
    Serial.begin(9600);
    
}
    
void loop() {
    
    Smol_Stepper.step(steps_per_rev);
    delay(1000);
    Smol_Stepper.step(-steps_per_rev);
    delay(1000);
    
}
